# ⚡ AI-Powered Dynamic Form Generator

An intelligent web platform that converts natural language prompts into fully functional, production-ready web forms using Generative AI.

---

## 🚀 Overview

This project automates form creation by leveraging AI to generate dynamic, scalable forms instantly. It eliminates manual effort, enhances productivity, and delivers a seamless user experience for data collection workflows.

---

## ✨ Key Features

* 🤖 **AI-Based Form Generation**
  Convert plain English prompts into structured, dynamic web forms

* ⚡ **Real-Time Form Rendering**
  Instantly generate and preview forms with responsive UI

* 🔒 **Secure Authentication System**
  Integrated authentication using NextAuth.js, JWT, and bcrypt

* 📊 **Analytics Dashboard**

  * Response tracking
  * Data visualization
  * CSV export for analysis

* 🧠 **Scalable Backend Architecture**
  Built using modular APIs with optimized database queries

* 🎯 **User-Centric Design**
  Clean UI with Tailwind CSS for smooth interaction and accessibility

---

## 🛠️ Tech Stack

* **Frontend:** Next.js, React.js, Tailwind CSS
* **Backend:** Next.js API Routes
* **Database:** MongoDB, Mongoose
* **Authentication:** NextAuth.js, JWT, bcrypt
* **AI Integration:** Generative AI APIs
* **Other Tools:** Axios, Git

---

## ⚙️ How It Works

1. User enters a prompt (e.g., *"Create a job application form"*)
2. AI processes the prompt and generates structured schema
3. Dynamic form UI is rendered in real-time
4. Users submit responses
5. Data is stored and visualized in the analytics dashboard
